# Movie Catalog Web Application

A full-stack movie catalog web application built using **React.js** and **Firebase**.  
The app allows users to browse movies, view details, add reviews, and manage authentication.

---

## Features
- User authentication (Login & Signup)
- Browse and view movie details
- Add and manage movies
- Post and view reviews
- Responsive UI with reusable components
